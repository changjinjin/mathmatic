一、外观模式简介
外观模式是设计模式中结构型模式的一种。它隐藏了系统内部复杂的实现，向客户端提供一个可以访问系统的接口。这种设计模式涉及到一个单一的类，该类提供了客户端请求的简化方法和对现有系统类方法的委托调用。
外观模式非常常见，例如 JAVA 的三层开发模式就使用了外观模式的思想。
